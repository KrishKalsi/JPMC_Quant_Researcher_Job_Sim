# JPMC_Quant_Researcher_Job_Sim

## Overview

This repository contains my work from the **J.P. Morgan Quantitative Research Virtual Experience Program**. The simulation provided hands-on exposure to real-world quantitative finance problems, focusing on data analysis, statistical modeling, and financial decision-making.

The project demonstrates my ability to apply mathematical and statistical techniques to solve practical problems in a financial context, similar to what is expected in a quantitative research role.

## Key Objectives

* Analyze financial datasets and extract meaningful insights
* Apply statistical and mathematical models to real-world scenarios
* Build data-driven solutions for pricing and risk-related problems
* Communicate results clearly and effectively

## Skills & Tools Used

* **Programming:** Python
* **Libraries:** NumPy, Pandas, Matplotlib, scikit-learn, scipy
* **Concepts:**

  * Statistical Analysis
  * Time Series Analysis
  * Data Cleaning & Preprocessing
  * Financial Modeling
  * Hypothesis Testing
